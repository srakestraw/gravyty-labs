# Components

Components are reusable UI elements built on top of the foundations.

## Structure

Components follow MUI patterns and conventions while incorporating custom Gravyty Labs styling.

## Component Categories

- **Inputs** - Form controls, text fields, selects, checkboxes, etc.
- **Navigation** - Buttons, links, menus, tabs, etc.
- **Feedback** - Alerts, dialogs, snackbars, progress indicators
- **Data Display** - Tables, lists, cards, badges, etc.
- **Layout** - Containers, grids, dividers, etc.
- **Surfaces** - Paper, cards, app bars, etc.

## Usage

All components should:
- Follow MUI component API patterns
- Use theme values from foundations
- Support both light and dark modes
- Be accessible and keyboard navigable
- Include proper TypeScript types
